## 5. Server Log Files

When this website is accessed, the server automatically records information in server log files.
The following information may be collected:

- IP address
- Date and time of request
- Requested URL
- Referrer URL
- User agent (browser and operating system)

### Purpose of Processing

This information is processed exclusively for technical purposes:

- maintaining the stability and security of the service
- detecting and preventing abuse
- troubleshooting technical issues
- applying rate limiting or blocking malicious requests

The collected log data is **not used to identify individual users or create user profiles**.

### Legal Basis

Processing is based on **Article 6(1)(f) GDPR** (legitimate interests), specifically the interest in operating a secure
and reliable website.

### Retention

Server logs are automatically deleted after **10 days**.

## 6. iCal Feed Generation

The service allows users to generate an iCalendar (iCal) feed URL based on a **MyAnimeList username**.
When an iCal URL is accessed:

- the server retrieves publicly available data associated with that username from MyAnimeList
- an iCal feed is generated from this information

The provided username forms part of the request URL and may therefore appear in server logs.

## 7. Responsibility for Submitted Usernames

The generated iCal URL contains the MyAnimeList username provided by the user.
Users should only generate feeds for usernames they control or have permission to use.

## 8. Caching

To reduce the number of external requests and improve service performance, responses retrieved from MyAnimeList may be
temporarily cached.

- Cache duration: **up to 1 day**
- Cached data is automatically deleted after this period.

No persistent user profiles or long-term records are created.
